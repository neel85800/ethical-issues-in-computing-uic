# Book Report

You selected a book, you read the book, and you will soon present to the class about the book (if you have not already)!

This exercise is a two-page book report (12pt font, double-spaced) intended to help you digest what you have been reading and **form your own thoughts** about the material.

The prompt is intentionally open-ended: tell us about the book. If you are looking for more structure, here are some pointers from 
[John Minniear](https://www.uwosh.edu/faculty_staff/minniear/Book%20Review%20European%20Geo.pdf), who suggests considering the following questions while reading:
  * What is the author’s main goal in writing this book? (Convince you of a specific position on a
controversy? Explain the background of an event? Raise awareness of a particular issue?)
  * What are the author’s main points?
  * What kind of evidence does the author provide to make these points? How convincing is
this evidence?
  * Did you find the book was well-written? (Easily understandable? Good style?)
  * What group of readers would find this book most useful (Lay people? Students? Experts in
the area?)

One standard structure for a book review includes the following components:
1. Introduction (one or two paragraphs)
  * Bibliographic information (author, title, date of publication, publisher, number of pages,
type of book)
  * Brief overview of the theme, purpose and your evaluation
2. Summary of the content
  * Brief summary of the key points of each chapter or group of chapters
  * Paraphrase the information, and use quotes sparingly
3. Evaluation and conclusion
  * Give your opinion about the book.
  * Was it easy to read or confusing?
  * Was the book interesting, entertaining, instructive?
  * Does the author support the arguments well?
  * What are the book’s greatest strengths and weaknesses?
  * Who would you recommend the book to?

### **Book Review Rubric**
I plan to use the following rubric, mostly borrowed from Behrooz Shirazi at [Washington State University](https://eecs.wsu.edu/~shirazi/cs402/book%20review%20rubric.html):

| **Category** | **Score**             | **Description** |
|:-------------|:--------------------|:----------------|
| **Mechanical (Sentence Level)** |   Excellent (10 pts)   | Writing is polished, concise, and free of grammar or mechanical errors. |
|  |   Good (8 pts)        | Writing is mostly clear and correct, with only minor errors. |
|  |   Adequate (6 pts)    | Some grammatical or mechanical issues remain, but they do not seriously hinder comprehension. |
|  |   Inadequate (0-4 pts)  | Sentences are difficult to read or understand due to frequent grammar or mechanical errors. |
| **Clarity (Paragraph Level)** |   Excellent (10 pts)   | The paper flows smoothly with strong transitions and clear paragraph structure. |
|  |   Good (8 pts)        | Paragraphs are clear and coherent with effective transitions. |
|  |   Adequate (6 pts)    | Organization is present but transitions are basic or uneven. |
|  |   Inadequate (0-4 pts)  | The paper lacks structure and logical flow between ideas. |
| **Organization (Whole Paper)** |   Excellent (10 pts)   | The paper is well-structured, logical, and engaging from start to finish. |
|  |   Good (8 pts)        | Ideas are presented logically and generally flow well. |
|  |   Adequate (6 pts)    | Basic organization is present, but the paper may wander or feel flat. |
|  |   Inadequate (0-4 pts)  | The paper is disorganized and difficult to follow. |
| **Explains the Author’s Principal Ideas and Conclusions** |   Excellent (10 pts)   | Clearly and completely explains the author’s main ideas and conclusions. |
|  |   Good (8 pts)        | Identifies and explains main ideas clearly but with minor gaps. |
|  |   Adequate (6 pts)    | Identifies main ideas but explanations are incomplete or unclear. |
|  |   Inadequate (0-4 pts)  | Fails to identify or explain key ideas and conclusions. |
| **Identifies and Justifies Points the Student Supports or Opposes** |   Excellent (10 pts)   | Clearly identifies and fully justifies positions with strong, logical reasoning. |
|  |   Good (8 pts)        | Clearly identifies positions and provides reasonable justification. |
|  |   Adequate (6 pts)    | Identifies positions but offers weak or incomplete justification. |
|  |   Inadequate (0-4 pts)  | Fails to identify positions or provide reasoning. |
